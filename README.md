# KKApply

KKApply is a student enrollment platform that guides students through the admissions pipeline. It manages and tracks mandatory events—like Open Days, Coding Challenges, and etc that users need to register for and successfully complete to enroll in the school.

## Tech Stack

- **Framework**: [SvelteKit](https://kit.svelte.dev/)
- **Database**: SQLite + [Prisma](https://www.prisma.io/) for types
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Runtime**: [Bun](https://bun.sh/)

## Getting Started

1. Install dependencies:
   ```sh
   bun install
   ```

2. Start the development server:
   ```sh
   bun run dev
   ```

3. Build for production:
   ```sh
   bun run build
   ```
